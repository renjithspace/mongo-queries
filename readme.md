# MongoDB queries
Realtime mongodb query playground

#### Serve
- Clone the repo
- Open `config.json` and set `mongodb_uri`
- Install dependencies
```
npm i
```
- Run socket server
```
npm run serve:socket
```
- Serve web panel
```
npm run serve:panel
```
